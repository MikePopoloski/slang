slang

SystemVerilog compilation, elaboration, analysis, tooling

slang is a C++ library and command-line tool for working with SystemVerilog, the industry-standard hardware description and verification language. It parses, type-checks, and elaborates any SystemVerilog design, providing the same kind of deep static analysis for hardware that a compiler like clang provides for software. slang is designed from the ground up as a reusable library, so it can be embedded in synthesis tools, simulators, linters, code editors, or any other tooling that needs to understand SystemVerilog source code.

Getting Started

CLI Tool
Download a pre-built binary from the GitHub releases page, or build from source. See the user manual to get started.
Python Scripting
Install pyslang from PyPI: pip install pyslang. See the tests and examples for how to make use of the API.
C++ Library
Link your CMake project to slang::slang. See the integration guide and developer guide for details.

Try slang live in your browser on the interactive playground.

Complete Language Support

slang has full support for parsing and elaborating all SystemVerilog language constructs, as specified in the 1800-2023 LRM.

See the language support page for more details.

Production Quality

  • The fastest and most compliant SystemVerilog frontend (according to the open source chipsalliance test suite)
  • Comes with a wide array of compatibility options, to support existing codebases built against other tools
  • Designed to scale efficiently to large commercial-grade codebases
  • Battle tested, fuzz tested, with high levels of unit test coverage

Powerful Static Analysis

slang performs full type checking, cross-module elaboration, semantic verification, and data-flow analysis.

slang comes with many built-in diagnostics, and has many ways to configure and control them.

Custom rules can be written on top of slang; for example, see the slang-tidy tool.

Designed as a Library

slang is designed from the ground up to be used as a library, with a clean and well-documented API for parsing, analyzing, and manipulating SystemVerilog code.

slang can be easily integrated into other tools, and comes with Python bindings for easy scripting and prototyping.

See the developer guide and API reference for more details.

Use Cases

Some examples of things you can use slang for:
  • Very fast syntax checking and linting tool
  • Dumping the AST of your project to JSON
  • Source code introspection via included Python bindings
  • SystemVerilog code generation and refactoring
  • As the engine for an editor language server
  • As a fast and robust preprocessor that sits in front of downstream tools
  • As a frontend for a synthesis or simulation tool, by including slang as a library

Notable Users

  • Yosys-slang -- Synthesis tool
  • CIRCT -- LLVM-based EDA framework, with slang as the official SystemVerilog frontend
  • slang-netlist -- Netlist analysis tool
  • slang-server -- Language server for editors (LSP)
  • pyslint -- Python-based linting framework built on pyslang
  • simview -- Design browser and waveform viewer
  • slang-lsp-tools -- Language server for editors (LSP)
